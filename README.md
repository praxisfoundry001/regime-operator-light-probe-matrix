# Regime Operator and Confound-Control Matrix for Light-Probe Experiments

## Subtitle

A Technical Appendix to *Optical Signatures of Structural Failure Regimes in Immiscible Fluids*

## What is this?

This is a technical appendix / experimental matrix for the light-probe protocol in immiscible fluids.

The main experiment proposes that independently classified immiscible-fluid failure regimes should produce distinguishable optical profile families when light is used as a probe. This appendix sharpens that prediction by asking what changes locally in the medium such that light behaves differently in each regime.

## Why should I care?

The appendix makes the experiment harder to overread.

It does not merely say that messy fluid optics may blur, scatter, move, repeat, or retain memory. It explains how five Structural Flow failure regimes — Dissolution, Shear, Drift, Echo, and Lock — should alter local optical admissibility differently.

It also names ordinary confounds, discriminants, and falsifiers so the test can lose.

## What is the claim?

The central technical claim is:

Each failure regime changes local optical admissibility differently, so each should be tested through a different optical profile, confound, discriminant, and falsifier.

Compressed:

Dissolution weakens readability.  
Shear fragments compatibility.  
Drift misaligns registration.  
Echo repeats without settlement.  
Lock over-retains history.

Those five local changes should not produce the same optical profile.

## What is this not?

This appendix does not report laboratory results.

It does not derive photon theory, Maxwell’s equations, quantum electrodynamics, full wave optics, full General Relativity, or numerical thresholds. It does not claim that optical data alone classify a regime. It does not claim that every real fluid region will express one pure regime.

It is a prediction-support and confound-control layer.
